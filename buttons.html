Buttons

This page showcases the button styles available in the design system.

Primary & Secondary States

Primary buttons highlight the main action (e.g., “Submit”), while secondary buttons support alternative actions (e.g., “Cancel”). Static versions are inactive, Active versions show focus or availability.

Interactive Buttons

Interactive buttons respond to user actions. Primary is for the main CTA (e.g., “Sign Up”), Secondary for alternatives (e.g., “Skip for now”).

Outline & Disabled

Outline buttons are used for optional actions (e.g., “Learn More”), while disabled buttons indicate unavailable actions (e.g., “Download” before data is ready).
