Typography

This page showcases the typography styles available in the design system.

Here there are all the titles, containers and showcases of each one. The explanation of the usage of each one is explained in the text under each title and subtitle. It uses the Poppins font, with a line-height of 24px (--line-height-sm variable), and a letter spacing of 0px (--letter-spacing-sm variable).

CSS


        .text {
            font-family: "Poppins", sans-serif;
            line-height: var(--line-height-sm);
            letter-spacing: var(--letter-spacing-sm);
        }
        
        .paragraph {
            color: var(--color-neutral-40);
            margin: 10px;
            padding-inline: 40px;
        }

Title Large

Subtitle Large

The Large Title is the main heading of a page. It introduces the page clearly and sets the focus. The Large Subtitle adds supporting context to the main title, helping explain the purpose of the page.

HTML


<section class="header header-landing">
  <hgroup class="hgroup">
    <h1 class="text title-lg">Title Large</h1>
    <h2 class="text subtitle-lg">Subtitle Large</h2>
  </hgroup>
  <p class="text paragraph">
    The Large Title is ...
  </p>
</section>

CSS


        .title-lg {
            font-size: 70px;
            font-weight: 500;
            color: var(--color-neutral-10); /*#FAFAFA*/
            line-height: var(--line-height-lg); /*64px*/
            letter-spacing: var(--letter-spacing-lg); /*-0.25px*/
            text-shadow: 4px 4px 1px rgba(0, 0, 0, 0.4); 
        }

        .subtitle-lg {
            font-size: 22.4px;
            font-weight: 400;
            color: var(--color-neutral-10); /*#FAFAFA*/
            line-height: var(--line-height-lg);  /*64px*/
            letter-spacing: var(--letter-spacing-lg); /*-0.25px*/
        }

Title Medium

Long descriptive subtitle of the page is in for medium title

The Medium Title is used for key sections inside a page. It helps organize content and guide the reader. The Small Subtitle under a medium title provides extra detail or explanation when needed.

Paragraphs

A Medium Title can also be used alone to introduce a paragraph or highlight a block of text. This makes the paragraph stand out as a distinct part of the content.

Small Titles are used for subsections or smaller content blocks. They can be styled in different colors (orange, blue, neutral) to match the design theme or highlight specific areas. A Small Subtitle can be added to give short supporting information and help readers understand the section quickly.

Paragraphs

Standard paragraphs are used for body text. They keep content clear and easy to read.

The Semibold Paragraph adds emphasis to important points within the text.

The Bold Paragraph is used for strong emphasis, quotes, or highlights that need attention.

Long paragraphs outside containers are useful for captions, notes, or secondary information that supports the main content.

Thin is the text label for users, used for messages and posts – minimal emphasis.
